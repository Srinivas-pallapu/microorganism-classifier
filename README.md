# 🔬 Microorganism Image Classifier

An AI-powered web application that classifies microscopic microorganism images using Deep Learning.

## 🚀 Live Features

✅ Image Classification  
✅ Video Analysis  
✅ Live Webcam Capture  
✅ Confidence Score Charts  
✅ Top 3 Predictions  
✅ Modern Streamlit UI  

---

## 🧠 Classes Detected

- Amoeba
- Euglena
- Hydra
- Paramecium
- Rod Bacteria
- Spherical Bacteria
- Spiral Bacteria
- Yeast
