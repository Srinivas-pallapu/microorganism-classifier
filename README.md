# 🔬 Microorganism Image Classifier

An **AI-powered web application** that classifies microscopic microorganism images using **Deep Learning**.

🌐 **Live App:** https://microorganisms-classifier.streamlit.app/

---

## 🚀 Features

✅ Image Classification
✅ Video Analysis
✅ Live Webcam Capture
✅ Confidence Score Charts
✅ Top 3 Predictions
✅ Modern Streamlit UI

---

## 🧠 Classes Detected

* Amoeba
* Euglena
* Hydra
* Paramecium
* Rod Bacteria
* Spherical Bacteria
* Spiral Bacteria
* Yeast

---

## 🛠️ Technologies Used

* Python
* Streamlit
* TensorFlow / Keras
* OpenCV
* NumPy

* Plotly
